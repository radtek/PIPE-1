Oracle Database

The Production Information PanCIM Exporter program reads and writes job data to Oracle database tables. The Oracle instance, user name and password data used for this connection is supplied in the InfoPlus.21 OracleJobManager record.

The PIPE program uses several DSNs that must be setup on the local server. (see Data Set Names) The Oracle drivers for ODBC must be installed if not available when configuring these DSN connections. Drivers can be downloaded directly from the Oracle web site if needed.
