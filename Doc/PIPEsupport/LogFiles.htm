Log Files

The Production Information PanCIM Exporter program writes information into a log file for many events. The name of this file is PIPE.LOG. It is located in the SETCIMLOGS folder and can be read with any ASCII editor such as notepad. Unlike the typical InfoPlus.21 "out" files, this log file is closed after each entry written which allows the viewing all current data without the need to stop the program. It is a good resource for troubleshooting problems associated with TSK_PIPE.

Note: In order to keep all the log files from growing very large, they are automatically renamed every 24 hours with the date of the month appended to the end of the file name. The PIPE program will create a new file if the PIPE.LOG file name does already not exist.
