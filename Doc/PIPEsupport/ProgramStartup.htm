Program Startup

The Production Information PanCIM Exporter (TSK_PIPE) is started through the InfoPlus.21 Manager similar to most other PanCIM programs with a few exceptions. The PIPE program should be started with the "WC" command line argument in Wiring Components and Singapore servers. All other locations will not need any command line argument to be supplied.

The WC argument instructs the PIPE program to decode several InfoPlus.21 records that are only used in WC and PSPL locations currently.

It is also recommended that the Auto Restart option be used with maximum retries set to 10 or more and a retry interval of 60 seconds.
