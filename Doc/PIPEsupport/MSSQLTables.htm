MSSQLTables

There are currently four tables in the MSSQL ProductionHist database required for the Production Information PanCIM Export program.

Database:ProductionHist

Table

Description

EqptHist

Contains production history information related to auxilary equipment

JobHist

Contains job history information from Oracle as well as some PanCIM collected data.

JobPerf

Contains some performance information collected by PanCIM related to the Oracle job.

MfgSystemInfo

Contains some static information related to molds when joined to other tables. Used for various reporting functions.

 

Database:EquipmentPerf

Table

Description

BaggerPerf

Contains bagger performance data (bagger cycles, fault counts, etc) associated with the Oracle job.

InserterPerf

Contains inserter performance data (inserter cycles, fault counts, etc) associated with the Oracle job.

LoaderPerf

Contains loader performance data (loader cycles, fault counts, etc) associated with the Oracle job.

MoldMachPerf

(not currently used)

 

Database:EquipmentRepair

Table

Description

BaggerRepairAction

Contains the repair action text associated with the InfoPlus.21 integer codes.

BaggerRepairHist

Contains bagger repair information collected by PanCIM.

BaggerRepairItem

Contains the repair item text associated with the InfoPlus.21 integer codes.

MoldRepairAction

Contains the repair action text associated with the InfoPlus.21 integer codes.

MoldRepairHist

Contains mold repair information collected by PanCIM.

MoldRepairItem

Contains the repair item text associated with the InfoPlus.21 integer codes.

See the Field Definitions topic for data definitions of the fields in these tables.
