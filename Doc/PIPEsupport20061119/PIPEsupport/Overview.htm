Overview

The Production Information PanCIM Exporter (TSK_PIPE) application's main function is to update the PanCIM database with specific information relating to Oracle job packet data. It also provides some basic production data to Oracle such as cycle counts and machine run times when jobs are complete.

This interface program also updates designated Oracle tables with necessary information regarding all Wiring Components Division Material Delivery System data. Most of this MDS data is collected and uploaded to Oracle automatically. (Wiring Components Division only)

There are other relational databases and tables that receive data from PanCIM through this interface. The following document provides details necessary for support and explains some of the more important functions. This document assumes the reader is somewhat familiar with InfoPlus.21 databases and terminology.
