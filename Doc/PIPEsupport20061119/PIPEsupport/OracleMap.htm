OracleMap Record

The PIPE program creates records in the InfoPlus.21 database using OracleJobDef definition in all divisions that currently use the PanCIMSE system. (all except WC and PSPL) The OracleMap record is used by the program to determine how to relate the Oracle field data to InfoPlus.21 fields and how to configure the OracleJob records when creating them.

 

Field Name

Description

Oracle_Field

Contains the field names used in the Oracle database table APPS.PANDUIT_WIP_PANCIM_JOB_V

IP21_Field

Equivalent InfoPlus.21 database field name to associate with the Oracle field.

Occurrence_Num

Occurrence number to use when adding label text into the new record defined by OracleJobDef. A value of zero indicates the fixed area of the record.

Data_Type

Indicates which data type repeat area to use for the label text.

Field_Label

Used by the local System Effectiveness program.

Use

Flag to indicate if data should be included when creating the new record defined by OracleJobDef.

 

IMPORTANT: A valid OracleJobDef record name must be present if the Use field is set to YES and the OCCURRENCE_NUM field value is 0.
