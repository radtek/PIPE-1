MSSQL Database

The Production Information PanCIM Export program requires a connection to three Microsoft SQL Server databases. These databases require permissions for local MSSQL login called JobHistInterface. The password for this MSSQL account will be supplied as needed.

 

Database

Description

EquipmentPerf

Contains several tables for storing various auxiliary equipment performance data.

EquipmentRepair

Contains several tables for storing repair data on various equipment. (molds, baggers, loaders, inserters)

ProductionHist

Contains four tables for storing production job data supplied by Oracle and PanCIM.

See MSSQL Tables for additional information on the tables in these databases.
